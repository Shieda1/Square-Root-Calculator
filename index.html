What is the square root?
The square root of a specific number is a value that when multiplied by itself gives the number.
How to calculate square root?
To calculate square root you should use the following formula:
x = (y)^1/2
Example: Find the square root of 4?
Using the square root formula the answer will be:
x = (y)^1/2
x = (4)^1/2
x = 2
Why use square root?
The square root has a major use in the formula for roots of a quadratic equation, quadratic fields and rings of quadratic integers.
